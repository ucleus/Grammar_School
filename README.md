## üìÇ Content Categories
The curriculum focuses on essential transition words and phrases for academic writing:
*   **Introduction** (e.g., "In this essay...", "The central theme...")
*   **Cause & Result** (e.g., "Therefore...", "Consequently...")
*   **Comparison** (e.g., "However...", "On the contrary...")
*   **Emphasis** (e.g., "Above all...", "Obviously...")
*   **Time & Order** (e.g., "First of all...", "Finally...")
*   **Conclusion** (e.g., "In conclusion...", "To sum up...")
